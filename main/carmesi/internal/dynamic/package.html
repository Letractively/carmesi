Internal classes of Carmesi for dynamic controller.

Beware of using classes in this package because could be changed without previous notification.

A controller must have exactly one public method. Parameters must obey one of the follow constraints:

  • Be one of the automatic injectable types: HttpSession, HttpServletRequest, HttpServletResponse, ServletContext.
  • Have one of the attribute annotations: RequestAttribute, SessionAttribute or ApplicacionAttribute
  • Have one of the parameter annotations: RequestParameter, ContextParameter.
  • Annotated with CookieValue.
  • Annotated with RequestBean.
